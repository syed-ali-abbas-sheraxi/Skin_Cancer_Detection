# Skin Cancer Detection

## Overview

A deep learning-based skin cancer detection system developed using the HAM10000 Kaggle dataset.

The project uses Convolutional Neural Networks (CNN) to classify different types of skin lesions from dermoscopic images.

## Features

- Image Preprocessing
- Data Augmentation
- CNN Model
- TensorFlow/Keras
- Performance Evaluation

## Dataset

HAM10000 Dataset

## Technologies

- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- OpenCV

## Results

The trained CNN model classifies skin lesions with high accuracy and provides performance evaluation using standard classification metrics.
